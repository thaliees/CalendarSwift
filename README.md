## Implement CocoaPods and Pod JTAppleCalendar
#### Install CocoaPods.
$ sudo gem install cocoapods

#### Create Podfile within of project
$ pod init

#### Modify Podfile
pod 'JTAppleCalendar'

#### Install Pod
$ pod install
